<br/>QUERY - Ключ запроса
<br/>MAX_PAGES - Кол-во страниц для парсинга
<br/>RETRIES - Кол-во попыток получения ответа от сервера
<br/>DELAY_RANGE - Диапазон ожидания между запросами (случайный, от x до y)

<br/>def get_page(page) - Получаем данные на странице
<br/>def get_basket(vol) - Определяем номер сервера по артикулу(id) без 5 разрядов
<br/>def get_card(id) - Получаем card.json, хранящий дополнительную информацию, описание и характеристики
<br/>def format_card(card) - Форматируем card.json
<br/>def get_images(id, count) - Создаём ссылки на изображения
<br/>def get_detail(id) - Получаем детальную информацию конкретной карточки товара
